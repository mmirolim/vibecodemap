Interactive 3D code city for Uzumtools. Far zoom shows visibly coded districts and an orthogonally routed road network with separately allocated district gates. Middle zoom shows component buildings with broad flush review bands, explicit input and output ports, persistent-state labels, bounded local feeders, and security-review markers. Near zoom focuses the selected component and its strongest routes. From bottom to top, bands encode coverage gap, complexity, mapped coupling, and direct mutating effects. Security marker size encodes severity and opacity encodes confidence; a marker is an audit lead, not proof of a vulnerability. By default roads omit static import and type dependencies. Road lane color identifies the selected relation family, segmentation identifies asynchronous or callback execution, width identifies aggregate mapped strength, and arrows identify direction. Keyboard movement is time-based and changes only the camera. No moving traffic is shown because runtime traces have not been observed.
