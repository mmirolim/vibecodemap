# VibeCodeMap product site

Public OpenAI Build Week landing page and no-install demo for VibeCodeMap.

The generated `public/demo.html` and `public/demo.view.json` are produced from
the repository-owned self-map under `../.vibecodemap/`. They are committed here
as deliberate judge-facing demo artifacts; ordinary generated maps remain
ignored.

```bash
npm ci
npm run dev
npm run build
```

The source project is documented in the repository root [README](../README.md).
